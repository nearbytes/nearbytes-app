Nearbytes storage location

This folder is being used by Nearbytes as a shared storage location for encrypted blocks and append-only space history.

You can leave this file in place. Nearbytes uses it to discover the location automatically.

Learn more at https://anymatix.github.io/nearbytes/.

On macOS, if you have just copied Nearbytes into Applications, you may need:

xattr -dr com.apple.quarantine "/Applications/Nearbytes.app"
